# HoneyTraveler Hugo Data Directory

This folder stores structured data files used by Hugo templates and partials.

Typical contents:
- `newsfeeds.csv` — external apiculture news feeds
- `research_sources.csv` — verified domain-level research sources
- `vegetal_honeys.csv` — cultural/folk syrup index
- `mead_index.csv` — fermented honey beverages (planned)

All CSVs or JSON files here are authoritative and sync from the iCloud “3_Data_Indices” folder.
